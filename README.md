---

 * Project Overview
The **Royal Public School Management System** is a full-stack application that helps manage:

- Student records
- Teacher management
- Attendance tracking
- Fees management
- Admin dashboard

---

 🛠️ Tech Stack
- **Frontend:** Flutter
- **Backend:** Django (Python)
- **Database:** SQLite / PostgreSQL
